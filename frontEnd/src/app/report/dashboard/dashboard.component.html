Deleted Documents Report

This report displays a list of all of the documents that have been deleted by each user for a specific date range. It contains the date deleted and the value of each index field for the given document. It can be filtered by date, and by user.


Workflow Report

This report displays a list of all of the documents that have been routed through the workflow process by each user for a specific date range. It can be filtered by date, by status, and by user.
